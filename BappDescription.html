Permitter is an extension that automatically tests for authorization issues by replaying requests from proxy history or the site map across multiple configured roles, using regex-based session token substitution to detect improper access controls.

Features

  • Tests authorization across multiple roles, each defined with a name and one or more regex patterns that swap session identifiers in requests
  • Pulls test targets from proxy history or the site map, with scope definable by target selection or regex
  • Optional unauthenticated access testing, disabled by default, to check endpoints without any session token
  • Endpoint exclusion list and configurable per-request delay to control test behavior
  • Deduplication of endpoints by default, with an option to use the full history when complete coverage is needed
  • Exports results as CSV or as HTML with embedded request and response data for detailed review
  • Save and load configuration to reuse role and scope settings across testing sessions

Usage

  1. Open the Permitter tab and navigate to the Scope section.
  2. To use proxy history targets, select Refresh Targets and choose a target from the list. Alternatively, define scope using a regex pattern.
  3. Add any endpoints to exclude from testing, and set the request delay if needed (default is 10 milliseconds).
  4. Navigate to the Roles section and add each role by entering a name and a regex pattern that matches and replaces the session identifier for that role, then select Save.
  5. Repeat for each additional role. Roles appear in the Roles: selector and can be modified after being added.
  6. Optionally check Include Unauth to also test endpoints without any session token, and check Use Entire History to disable deduplication.
  7. Select either Use Proxy History or Use Site Map to begin testing. Results populate in the Permitter results panel.
  8. Export results using the CSV or HTML option. For HTML exports, select Show Request/Response on any result to review the full request and response detail.
